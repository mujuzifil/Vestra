# Stage 18.3 — Checkout Flow Report

## Registered Customer Flow

```
Cart
  ↓
Checkout
  ↓
Customer Information (read-only from profile)
  ↓
Shipping (select saved address or enter new address)
  ↓
Payment (method + shipping method)
  ↓
Review
  ↓
Place Order
  ↓
Order Confirmation
  ↓
Payment (Stage 18.4)
```

### Registered customer details
- Name, email, phone auto-filled from `useAuth().user`.
- Saved addresses loaded from `/auth/addresses`.
- New address can be entered and optionally saved.

## Guest Customer Flow

```
Cart
  ↓
Checkout
  ↓
Customer Information (name, email, phone)
  ↓
Shipping Address
  ↓
Payment (method + shipping method)
  ↓
Review
  ↓
Auto-register account
  ↓
Merge guest cart
  ↓
Place Order
  ↓
Order Confirmation
  ↓
Payment (Stage 18.4)
```

### Guest account creation
- Guest enters name, email, phone.
- On "Place Order", frontend generates a secure random password.
- Calls `/auth/register`.
- On success, calls `mergeGuestCart()` to move localStorage cart to server.
- Calls `/checkout` to create the order.
- If the email already exists, user is prompted to log in or use a different email.
- Guest checkout info is persisted in localStorage until order success.

## Address Collection

### Fields
- Recipient name
- Recipient phone
- Country (default: Uganda, read-only)
- Region / State
- District / County
- City / District
- Street address
- Postal code
- Nearest landmark
- Delivery instructions

### Validation
- Recipient name required (min 2 chars).
- Phone required (min 7 chars).
- City required.
- Street address required (min 5 chars).

## Payment Methods
- Cash on Delivery (COD)
- MTN Mobile Money
- Airtel Money
- Card Payment

## Shipping Methods
- Standard Delivery — 3–5 business days, free
- Express Delivery — 1–2 business days, UGX 15,000
- Pickup — placeholder (disabled)
