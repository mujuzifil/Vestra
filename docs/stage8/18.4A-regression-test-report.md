## Scope

Verify that the admin portal migration does not break any existing production functionality.

## Website Tests

| Test | Command | Expected |
|------|---------|----------|
| Homepage loads | `curl -I https://vestradetergents.com` | HTTP 200 |
| Products page | `curl -I https://vestradetergents.com/products` | HTTP 200 |
| Shopping cart | Browser: add item to cart | Cart updates |
| Checkout | Browser: proceed to checkout | Checkout page loads |

| Test | Command | Expected |
|------|---------|----------|
| Health endpoint | `curl https://api.vestradetergents.com/api/v1/health` | JSON success |
