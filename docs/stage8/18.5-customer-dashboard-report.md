# Stage 18.5 — Customer Dashboard Report

## Dashboard Widgets

### Order Statistics

| Stat | Color | Icon |
|------|-------|------|
| Total Orders | Dark | Package |
| Pending Payment | Amber | CreditCard |
| Processing | Blue | Truck |
| Completed | Green | CheckCircle2 |

### Quick Actions

| Action | Destination | Color |
|--------|-------------|-------|
| Continue Shopping | `/products` | Green |
| Track Order | `/track` | Navy |
| View Orders | `/account/orders` | Blue |
| Update Profile | `/account/settings` | Slate |

### Recent Orders

- Displays up to 5 most recent orders.
- Each row shows invoice number, date, status badge, total, and contextual action buttons:
  - **Pay** — shown when payment is pending.
  - **Track** — shown when order is shipped or delivered.
  - **View** — always shown.

### Account Navigation

- My Orders
- Addresses
- Settings
- Sign Out

## Mobile Layout

- Stats grid collapses to 2 columns on small screens.
- Quick actions grid collapses to 2 columns.
- Recent order rows stack vertically on mobile.
- Sidebar navigation remains accessible.
