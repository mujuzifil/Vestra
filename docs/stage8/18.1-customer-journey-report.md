# Stage 18.1 — Customer Journey Report

## Overview
Production acceptance testing revealed that the website was funnelling retail shoppers into the distributor application workflow. This report documents the redesigned separation of three independent business journeys.

## Journey 1 — Retail Customer (Primary)

```
Home
  ↓
Products
  ↓
Category / Search
  ↓
Product Details
  ↓
Add to Cart / Buy Now
  ↓
Shopping Cart
  ↓
Checkout
  ↓
Payment
  ↓
Order Confirmation
  ↓
Order Tracking
```

### Key decisions
- Shopping is now the default path from every product touchpoint.
- Product cards and product detail pages expose **Add to Cart** and **Buy Now** actions.
- Distributor content is moved to a secondary link; it no longer replaces purchasing actions.
- Cart drawer and checkout pages already exist and are now reachable from product pages.

## Journey 2 — Distributor Application (Independent)

```
Home / Nav
  ↓
Become Distributor
  ↓
Information & Benefits
  ↓
Application Form
  ↓
Application Submitted (success page)
  ↓
Application Review (5–7 business days)
  ↓
Approval
  ↓
Distributor Dashboard (future)
```

### Key decisions
- Distributor application is accessed only through the **Become a Distributor** nav item and secondary footer/CTA links.
- On submission the user is taken to a dedicated success page with a reference number.
- The application form is no longer shown after successful submission.

## Journey 3 — Corporate / Bulk Orders (Independent)

```
Home / Nav
  ↓
Bulk Orders
  ↓
Request Quote
  ↓
Sales Review
  ↓
Quotation
  ↓
Approval
  ↓
Invoice
  ↓
Payment
```

### Key decisions
- A new **Bulk Orders** page provides a quotation request form.
- Wholesale buyers never enter the retail checkout.
- The form is client-validated and shows a success confirmation; backend integration is reserved for a future stage.

## Summary
All three journeys are now visually and logically separated. Retail shopping is the primary experience, while distributor and bulk workflows are optional, clearly labelled business paths.
