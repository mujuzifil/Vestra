# Stage 18.4A — Domain Architecture Report

## Current Architecture

| Subdomain | Destination | Paths |
|-----------|-------------|-------|
| `https://vestradetergents.com` | Next.js frontend | `/` (all customer pages) |
| `https://api.vestradetergents.com` | Laravel backend | `/api/v1/*`, `/admin`, `/storage/`, `/livewire/`, `/` |

Before this stage, the Filament admin panel was served as a path under the API domain:

```
https://api.vestradetergents.com/admin
```

## Target Architecture

| Subdomain | Destination | Paths |
|-----------|-------------|-------|
| `https://vestradetergents.com` | Next.js frontend | `/` (all customer pages) |
| `https://api.vestradetergents.com` | Laravel backend | `/api/v1/*`, `/storage/` only |
| `https://admin.vestradetergents.com` | Laravel backend | Filament admin at `/` |

After this stage, the Filament admin panel is served from its own subdomain:

```
https://admin.vestradetergents.com
```

## Routing Changes

### Nginx Virtual Hosts

The nginx template `nginx/conf.d/vestra.conf.template` now defines three vhosts:

1. **Frontend vhost** (`${APP_DOMAIN}`, `www.${APP_DOMAIN}`)
   - Proxies all traffic to the Next.js container.
   - Unchanged.

2. **API vhost** (`${API_DOMAIN}`)
   - Proxies `/api/v1/*` to Laravel.
   - Proxies `/storage/` to Laravel.
   - Redirects `/admin` and `/admin/*` to `https://${ADMIN_DOMAIN}` with HTTP 301.
   - No longer serves Filament.

3. **Admin vhost** (`${ADMIN_DOMAIN}`)
   - New dedicated vhost listening on 443 for `admin.vestradetergents.com`.
   - Proxies `/`, `/livewire/`, `/build/`, and `/storage/` to Laravel.
   - Serves Filament from the domain root.

### Filament Configuration

`backend/app/Providers/Filament/AdminPanelProvider.php`:

- Added `->domain(config('app.admin_domain'))`.
- Changed `->path('admin')` to `->path('')`.

`backend/config/app.php`:

- Added `'admin_domain' => env('ADMIN_DOMAIN', '')`.

## Environment Variables

New and updated variables:

| Variable | Purpose |
|----------|---------|
| `ADMIN_DOMAIN` | Hostname for the admin portal (e.g. `admin.vestradetergents.com`) |
| `SANCTUM_STATEFUL_DOMAINS` | Now includes `admin.vestradetergents.com` |
| `CORS_ALLOWED_ORIGINS` | Now includes `https://admin.vestradetergents.com` |
