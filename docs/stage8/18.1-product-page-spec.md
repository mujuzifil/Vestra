# Stage 18.1 — Product Page Specification

## File
`frontend/app/products/[slug]/product-page-client.tsx`

## Layout

### 1. Page Hero
- Title: product name
- Subtitle: short description
- Breadcrumb: `Products > Product Name`

### 2. Product Detail Grid (2 columns on desktop)

#### Left Column — Product Gallery
- Main image display
- Thumbnail strip (when multiple images exist)
- Placeholder fallback

#### Right Column — Product Information
- Category badge
- Product name (H1)
- Price (large, prominent)
- Meta row:
  - SKU
  - Availability badge (`In Stock (n)` / `Out of Stock`)
- Description
- Benefits list (if available)
- Key Features tags (if available)
- Specifications table (if available)
- Reviews summary (if reviews exist)

### 3. Commerce Action Block
- **Quantity selector** (default 1, min 1)
- **Add to Cart** button
  - Primary navy gradient style
  - Calls `useCartContext().addItem(product.id, quantity)`
  - Loading and added states
- **Buy Now** button
  - Green gradient style
  - Adds to cart then navigates to `/checkout`
- Secondary prompt:
  - Links to `/bulk-orders` (wholesale quote)
  - Links to `/distributor` (become a distributor)

### 4. Reviews Section
- Customer reviews list
- Review submission form

### 5. Recently Viewed Section
- LocalStorage-based list of recently browsed products
- Displays only when items exist

### 6. Related Products Section
- Up to 3 products from the same category
- Card with image, name, price

### 7. Bottom CTA
- Heading: "Need wholesale or corporate pricing?"
- Primary button: `/bulk-orders`
- Secondary button: `/distributor`

## Commerce placeholders reserved
The following are not implemented yet but have reserved UI locations:
- Stock availability badge (uses `product.stock_quantity`)
- Promotions / coupons (reserved near price and checkout summary)
- Payment methods (reserved in checkout)

## Notes
- The previous "Become a Distributor" primary CTA has been removed from the product detail page.
- Distributor content is now secondary and never replaces purchasing actions.
