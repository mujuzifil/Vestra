# Stage 18.4A.1 — Regression Validation Report

## Scope

This report covers regression testing after correcting the `/admin/*` redirect behaviour. No customer-facing UI, checkout, payment, order, or tracking logic was changed; only nginx routing and the admin redirect were modified.

## Test Environment

- Production VPS IP: `187.77.84.119`
- Website: `https://vestradetergents.com`
- API: `https://api.vestradetergents.com`
- Admin portal: `https://admin.vestradetergents.com`
